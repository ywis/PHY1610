The main file hw1.cpp contains the main function that could print a 3-column, 100-row array in a text file "hw1.txt". The first column is x ranging from -5 to 5, the second column is cos(2x), and the third column is sin(3x).
Two functions contained in this file are for computing cos(2x) and sin(3x).

the compilation command is following this:
g++ -std=c++11 -o hw1 hw1.cpp -g -O2

the run command is
./hw1
